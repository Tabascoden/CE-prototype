# AGENTS.md

## Главный процесс (обязательно)
- Всегда работай через Pull Request (ветка -> PR).
- Проверка только через Vercel Preview. Не проси меня запускать localhost.
- В PR обязательно добавь: что сделано + как проверить (через Preview) + что изменилось (кратко).

## Качество
- Если есть lint/test/build — запусти и исправь ошибки до сдачи.
- Делай изменения небольшими и понятными.

## Безопасность
- Никогда не добавляй секреты/ключи в репозиторий.
- Если нужны env-переменные — перечисли их, чтобы я добавил в Vercel (Environment Variables).

## UI/UX по умолчанию
- Должно нормально выглядеть на телефоне и на компьютере.
- Формы: валидация (минимальная), понятные сообщения об ошибках.
